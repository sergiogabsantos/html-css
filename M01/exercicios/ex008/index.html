Negrito e Italico

Seguindo a 'forma' semâtica

A tag para deixar em NEGRITO (de forma semantica) é: <strong>

A tag para deixar em itálico é <em >

Marcador de texto

pode marcar um texto, como a caneta, usando a tag <mark>

Texto de deletado

Da para "excluir" um texto ou frase com a tag: <del>

Ênfase em texto

Para da enfâse num texto se usa a tag <ins>

Texto sobrescrito

Um exemplo de dexto sobrescrito é: x20+30. É usando a tag <sup>

Texto subescrito

Exemplo de um texto subescrito é: H2O.É possivel isso usando a tag: <sub>
